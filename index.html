Make it easy for your WordPress posts to go viral

Simply Pinterest uses the Pinterest "pin it" button to help you promote Pinterest sharing on your blog. It's simple to configure your blog's default settings then customize on a per-post or per-image basis. No matter where the post is pinned from it will always credit the permalink (no more archive pages being pinned!) Using the best SEO practices this free plugin makes it obvious and simple to your visitors that they should share your great content on Pinterest!

Configure Simply Pinterest

Simply Pinterest is easy to configure. Set your blog's default setting in Setting > Pinterest Settings

Choose the color, positioning and design of the Pinterest "Pin it" button using the native Pinterest design which your visitors will recognize and engage with quickly.

Easily disable the button on pages or posts

If there is a page or post you want to disable the Pin it button over your images just check the Disable Pin It button for all images in this post checkbox on the edit post or edit page screen.

Change settings for specific images

If there is an image in a post you want to disable the button or change the settings on you can do that easily trough the WordPress Visual editor. Once the image has been added to your post click the pencil edit button and choose your settings for that image.

That's it!

Set your defaults and then just override for specific images
It's as easy as: set it and forget it!

Installation

FTP Upload

  1. Download the free plugin: Simply Pinterest
  2. Unzip or Extract the contents of the download
  3. Using an FTP client login to your hosting account and go to your WordPress installtion in the /wp-contents/plugin/ directory
  4. Upload the folder /simply-pinterest/ and all of it's contents to the plugin directory
  5. Login to your WordPress admin to activate and configure the plugin

Add Plugin Through Admin

  1. Download the free plugin: Simply Pinterest
  2. Login to your WordPress admin and go to Plugins > Add New the click the Upload Plugin button at the top of the page
  3. Click choose file and upload the zip file you downloaded in the first step
  4. Activate and configure the plugin through your WordPress admin

FAQ

Can I swap the button with my custom graphic?

No, there are other plugins that specialize in that functionality. Simply Pinterest is intended to stand out using Pinteret's signature "pin it" button; when something stands out to your visitors they are more likely to see it and click it than if it blended in with your blog's design.

How do I customize the default Pin description?

When you add an image to your post you can customize the "title" or "alt" text for the image through the WordPress admin, this text is what the plugin uses to pre-populate the Pin's description. This way you can customize your text depending on what image they click to pin. in Settings > Pinterest Settings you can automatically append content to the default description like: - from MyBlog.example.com

Does it work on mobile?

Yes there is a setting to always display the button when on a mobile device!

Will this work on my WordPress.com blog?

This plugin is designed for self hosted or WordPress.org blogs. Submit a feature request to suggest other blog platforms.

Did you find a bug?

If you think you've found a bug in the Simply Pinterest plugin here's what to do:

Go to the Github issues page to see if your bug has already been addressed. If you haven't found a solution you can report a bug by clicking "New issue" on that page. Please be as detailed as you can and provide a link to a working example, if possible. Submitting an issue requires you to register a free Github account but this is how the developer(s) keep any bugs, tasks and features organized.

You can also email your bug or issue to terri ann at simply pinterest dot com (remove spaces and replace dot and at) (if Github isn't your jam!)
